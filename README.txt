Tape Measure
------------

for Mac OS X, tested only on Lion

A tape measure app for mac, it lets you also get hex colours from mouse position 
and copies everything to the clipboard.

Drag your mouse to measure the width and height of something. Press the spacebar
to append to your clipboard whatever is showing in the bottom left. Use m and c
keys as described below to select the information you want.

Keys
----
[ h ] hide the menu
[ c ] toggle colour
[ m ] toggle mouse position
[' '] append to clipboard
[esc] quit (or click without dragging)

Shift-drag to do horizontal or vertical lines.
